## 简介
只是对接了自定义的后端框架，实现了用户管理，权限控制，简单 快速，没有多余功能
后台服务使用springCloud开发 https://github.com/kingwsi/base-cloud

项目下载和运行
----
- 安装依赖
```
yarn install
```

- 开发模式运行
```
yarn run serve
```

- 编译项目
```
yarn run build
```

- Lints and fixes files
```
yarn run lint
```
